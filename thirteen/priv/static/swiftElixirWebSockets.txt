    | 3. POST /api/tasks (Task B)    |------>| (Controller creates Task B, enqueues Job B)    |

    |                                |       |                                                |
    | 5. POST /api/tasks (Task C)    |------>| (Controller creates Task C, enqueues Job C)    |
    |                                |<------| 6. Returns {task_id: "C"}                      |
    |                                |       |                                                |
    |--------------------------------|       |------------------------------------------------|
    | 7. **Connect to WebSocket ONCE** |------>| Phoenix Channel (`UserSocket`)                 |
    |                                |       |                                                |

    | 9. Join Channel "task:B"       |------>| (Server authorizes and confirms join)          |
    | 10. Join Channel "task:C"      |------>| (Server authorizes and confirms join)          |
